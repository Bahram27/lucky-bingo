# 🎱 Lucky Bingo

Azərbaycanın ən yaxşı onlayn bingo oyunu!

## 🚀 GitHub Pages ilə Deploy

### Addım 1 — Repository Yarat
1. [github.com](https://github.com) → "New repository"
2. Ad: `lucky-bingo`
3. **Public** seç → "Create repository"

### Addım 2 — Faylları Yüklə
```
index.html
manifest.json
README.md
```

### Addım 3 — GitHub Pages Aktiv Et
1. Repository → **Settings**
2. Sol menyu → **Pages**
3. Source: **Deploy from a branch**
4. Branch: **main** → folder: **/ (root)**
5. **Save** bas

### Addım 4 — Link Al
Təxminən 2-3 dəqiqə sonra:
```
https://SƏNIN_ADUNUZ.github.io/lucky-bingo/
```
